Minecraft Sun & Shadow

Unity 3D | 太阳与影子科学实验

Unity 2022.3 LTS 3D 实时渲染 纹理贴图 天文模拟
GitHub 仓库 在线演示 (Web版)

快速开始

1. 安装 Unity Hub 和 Unity 2022.3 LTS

2. git clone https://github.com/guanwenjie928/minecraft-sun-shadow-unity.git

3. Unity Hub → Add → 选择项目文件夹

4. 打开 Assets/Scenes/MainScene.unity

5. Tools → Minecraft Sun Shadow → Setup Scene

6. 按 Play 运行
